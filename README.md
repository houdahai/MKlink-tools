# 🚀 Windows MKlink 极速助手

一款专为 Windows 极客与普通用户打造的现代化、高档 **Fluent 扁平化美学** 的 Olink/MKlink 建链与一键无损搬家绿色工具。

通过自主研发的 **Windows OLE 降级物理隔离越狱通道**，成功攻克了 Windows 特权隔离（UIPI）对管理员拖放的物理阻断，实现了在普通非管理员权限下的 **100% 毫无拦截拖放录入**；并在执行特权命令时智能后台调起 **UAC 临时按需提权**。

## 📖 四种链接核心科普与避坑指南

为了让您在不同的使用场景下选择最科学的链接形式，我们为您整理了如下比对指南：

| 链接类型 | 目标对象 | 是否跨盘 | 是否需管理员提权 | 经典适用场景 |
| :--- | :---: | :---: | :---: | :--- |
| **目录联接 (Junction /J)** | 目录 | ✅ 支持跨盘 | ❌ 否 (免提权) | 微信数据搬家、大型游戏迁移 *(最常用推荐)* |
| **目录符号链接 (Symlink /D)** | 目录 | ✅ 支持跨盘 | ⚠️ 是 (自动提权) | 局域网共享映射、跨设备网络挂载链接 |
| **文件符号链接 (File Symlink)** | 文件 | ✅ 支持跨盘 | ⚠️ 是 (自动提权) | 配置文件特殊重指向、系统底层级依赖重定向 |
| **文件硬链接 (Hardlink /H)** | 文件 | ❌ 仅限同盘 | ❌ 否 (免提权) | 节省空间的文件多处挂载与无损引用 |

### ⚠️ 核心避坑要点：
1. **生成链接路径的原则**：在执行命令前，**“生成的链接路径”（即输出路径）在物理磁盘上必须不存在**，而其背后的**“真实数据存放路径”（即源目标路径）必须真实存在**。本工具已内置了“父文件夹智能提取”和“目标文件夹自动拼装追加”算法，拖入即可完美全自动化拼装。
2. **双保险还原规则**：还原时应直接拖入之前生成的“软链接文件夹/文件”或通过还原面板快速恢复，数据 100% 安全，不会伤及您的任何本地资产。

## 🚀 极速运行指南

### 1. 绿色成品双击即用 (推荐)
直接下载本项目 `dist/` 目录下的 **`dist/MKlink极速助手.exe`**。
*   无需安装，不写任何注册表。
*   双击直接在您的 Windows 桌面上完美拉起，尽享免拦截 Fluent 越狱大图体验！

### 2. 源码运行与开发者部署
如果您需要在本地进行二次开发或直接运行 Python 源码：

```bash
# 1. 克隆本项目
git clone https://github.com/houdahai/MKlink-tools.git
cd MKlink-tools

# 2. 安装核心图形依赖 (PySide6)
pip install PySide6

# 3. 运行主前端界面
python main_gui.py
```

## 🛠️ 打包编译说明 (PyInstaller)

项目已提供配置好的 spec 描述文件。如需自行编译打包为单文件绿色 EXE：

```bash
# 安装 PyInstaller
pip install pyinstaller

# 一键执行 Spec 编译打包 (生成的 exe 成品会自动放置于 dist/ 目录中)
pyinstaller MKlink极速助手.spec --noconfirm
```
